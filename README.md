# Nuclear-MDP-Controller
IA Nuclear Control MDP ☢️ Este proyecto consiste en el diseño e implementación de un sistema de control inteligente para un reactor nuclear sometido a altos grados de incertidumbre. El objetivo principal es ajustar la potencia del reactor (estratificada en 100 niveles discretos) para que siga una curva de demanda energética variable en el tiempo.
